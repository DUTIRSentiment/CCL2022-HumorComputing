# CCL2022-HumorComputing

## New !
已更新评测数据集，于dataset文件夹中。train与friends.csv为训练集，task1_dev，task2_dev为两个任务的验证集，task1_test，task2_test为两个任务的测试集。测试集于9月15日公布，第二阶段周期为两周。

## 说明

为了方便各参赛队伍对任务二的结果进行评估，现在将提供任务二的各项子指标结果。
